# Примеры кода sockets

###### **тгк:** [BoCoder_Python](https://t.me/BoCoder_Python)

В этом репозитории собраны базовые примеры работы с TCP-сокетами.

Проект демонстрирует создание эхо-сервера, который ожидает подключения, принимает данные от клиента и отправляет их обратно.

## Содержание

* server.py - код для сервера.
* client.py - код для клиента.

## Как запускать?
**1. Клонируйте репозиторий и перейдите в папку проекта:**

Пример (Консоль):
```bash
git clone https://github.com/BoCoder17/sockets.git
```
```bash
cd sockets
```
**2. Запустите сервер:**

```bash
python server.py
```
Сервер запустится и будет ждать подключения клиента.

**3. После этого в другом окне запустите клиента:**
```bash
python client.py
```
Клиент подключится к серверу, передаст данные на сервер, получит ответ и напечатает его.

**4. Клиент и сервер завершат работу автоматически.**
